# Уязвимости:

*Критический фактор* - по своей природе ГСЧ может быть уязвим к периоду генерации случайного значения: период между генерациями - критический фактор.<br>
**Идеальный период** исходя из [[0](https://github.com/mshunko101/science/blob/main/%D0%9C%D0%BE%D0%BD%D0%BE%D0%B3%D1%80%D0%B0%D1%84%D0%B8%D1%8F/%D0%A2_%D0%90%D0%94%D0%90.md)] идеальный период ГСЧ зависит **внезапно** от средней продолжительности жизни на Земле, на форме этот параметр [**ОПЖ**](https://ru.wikipedia.org/wiki/Ожидаемая_продолжительность_жизни)[(**LE**)](https://en.wikipedia.org/wiki/Life_expectancy).<br>
<br>
*Форс-мажор* - исходя из принципов моей сумашедшей науки, энтропия и неупорядоченность может упорядочится в некоторый момент времени, в некоторой области - это фактор форс-мажор для криптографии и ЭЦП.
